Estrutura básica do HTML

O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web.

O HTML consiste basicamente em 3 partes:

  • Tag chamada html

    • Representa a raiz de um HTML. Assim, todos os outros elementos (filhos) devem ser descendentes desse elemento (pai).

    Obs. toda tag HTML há um sinal de menor (<), um sinal de maior (>) e uma palavra entre elas. Assim, forma-se o nome da tag.
    Obs. e quando há a barra (/) antes da palavra, significa que a tag está sendo fechada.
  • Tag chamada cabeça ou head

    • São instruções/informações que serão pré-carregadas antes de mostrar ao usuário. Assim, é na tag head que usa-se a tag title que é o título da página.

  • Tag chamada corpo ou body

    • Tudo o que estiver entre o “corpo”, ou seja, a tag body, são as coisas que irão aparecer para o “usuário.

O que são Tags?

{voltar}

As tags são comandos utilizados para informar o navegador a estrutura do site.Assim, é uma forma de classificar e organizar o conteúdo HTML.

Exemplos de tags:
  • < i > - deixa em itálico o texto;
  • < strong > - deixa em negrito o texto;
  • < u > - underline ou sublinhado;
  • < mark > - coloca marca texto;
  • < sub > - coloca marca de registrado;

Atributo das Tags

{voltar}

São configurações propriedades que se colocam na tag para que ela possa aparecer diferente no navegador. Há atributos que são mais globais géricas*, ou seja, todas as tags irão ter, mais também existem outros atributos que são específicos de cada tag.

Tags por Finalidade

{voltar}
  • Tag de texto

      • Principais tipografias para HTML:

    • Tags < h1 >,< h2 >, < h3 >...< h6 > – são tags voltadas para títulos, onde "h" significa header (cabeçalho).
    • Tag < p > – voltada para corpo do texto em si, ou seja, parágrafos.
    • Tag < blockquote > – utilizado para citação ou também pode ser utilizado para observação.

  • Tag de linha

    • Tag < hr /> - adiciona um traço de linha horizontal na página.

Listas Ordenadas e Não Ordenadas

{voltar}

Listas são recursos extremamente usados e é um importante recurso de HTML, pois permite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. Assim, há diferentes tipos de estrutura de listas.

Dentre elas, a diferença entre a lista ordenada (tag < ol >) e a lista não-ordenada (tag < ul >) está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

Obs. hoje, boa parte dos menus em HTML são feitos com listas.

Links

{voltar}

A palavra link significa navegação, assim, para internet é quando uma página se liga na outra através do acionamento de eventos como clique ou enter.

Há possibilidade também se criar links para navegação dentro da mesma página sem a necessidade de abrir outra guia pelo método chamado de âncora .
